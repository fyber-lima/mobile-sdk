# AdMob - adapter info

| Network | Adapter version | Third party SDK version | Fyber SDK version |
|:----------:|:-------------:|:-----------------------:|:------------:|
| AdMob | 2.1.0 | 7.0.0 | 7.0.3 + |

**Important:** 
>AdMob requires deployment target of 6.0 or higher

## Example parameters

* **name**: `AdMob`
* **settings**:
	* **SPAdMobInterstitialAdUnitId**
	* **SPAdMobIsCOPPACompliant**: `YES` | `NO`
